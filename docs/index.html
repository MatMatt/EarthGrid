🔑 Key Features

🧩

Content-Addressed

Every chunk identified by SHA-256 hash. Tamper-proof by design — corrupted or fake data is automatically rejected.

🔄

Auto-Sync

New data automatically propagates to all registered nodes. No manual intervention needed.

🐢

Idle Resources

Runs at lowest CPU and I/O priority (nice 19, ionice idle). Never competes with your other workloads.

🛰️

STAC Native

Built-in STAC catalog. Every item is discoverable, searchable, and interoperable with the EO ecosystem.

🧮

Built-in Processing

NDVI, NDWI, EVI, cloud masking, band math — process data where it lives, no data movement needed.

🌐

Federation

Beacons coordinate, data flows peer-to-peer. No single point of failure. Works across firewalls via HTTPS.

🚀 Quick Start

pip install earthgrid
earthgrid setup
earthgrid start --beacon https://mattiuzzi.zapto.org/earthgrid

Join the network in 3 commands. Storage, beacon URL, and sync are configured interactively.
